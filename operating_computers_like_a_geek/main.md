## 搭建动态网站：Django

### 实践：按照 tutorial 搭建 polls

Python/Golang 是当前微服务模式的主流开发语言，RESTful API 的开发其实大同小异。

* 开发简易的 web 应用
* python 语言运用
* 理解 web
* 进一步理解 http
* 理解 MVC 模式
* 理解服务器渲染和前端渲染的区别
